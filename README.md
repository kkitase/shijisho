# 想いを伝える、指示書。（Shijisho Generator）

テキストベースの修正指示と対象画像から、「想い」の伝わる視覚的な指示書を自動生成するアプリケーションです。
Gemini 2.0 Flash を活用し、高速かつ的確に修正箇所を特定します。

![Screen Shot](public/screenshot.png)

## 🚀 特徴

- **想いを汲み取る**: 単なる座標指定ではありません。自然言語の指示から「ここをどう直したいか」という意図を理解し、最適な位置に番号と矢印を配置します。
- **対話的な調整**: AI が生成した矢印は、マウスで自由にドラッグして調整可能。納得のいくまで「手ざわり感」のある操作で位置を変えられます。
- **豊かな表現力**:
  - 🖋 **フォント切り替え**: 明朝体や游ゴシック、ヒラギノ角ゴなど、指示書の雰囲気に合わせたフォントを選択。
  - 📏 **文字サイズ**: 視認性に合わせて、数値入力を介さずスライダーで直感的に調整。
  - 🎨 **色彩パレット**: 虹色（自動）や特定のカラー、さらには「誠実さ」を伝える黒まで、多彩なパレットから選択可能。
- **寄り添うデザイン**: 「ものさす」のブランドコンセプトを体現。余白を活かした静謐な UI が、修正指示という作業をクリエイティブな時間へと変えます。

---

## 📖 使い方

1. **対象を選ぶ**: 修正したい画像（デザイン案など）を選択、またはドラッグ & ドロップします。
2. **想いを記す**: 「ここをこうしてほしい」という内容を、いつもの言葉で入力してください。
3. **視覚化**: AI が指示を解釈し、即座に矢印と番号を描画します。
4. **細部を整える**: ドラッグで位置を微調整したり、フォントや色を変えてニュアンスを調整します。
5. **届ける**: 完成した指示書をダウンロードし、チームメンバーへ共有。あなたの想いが、まっすぐ正確に伝わります。

## 🛠 セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 環境変数の設定

`.env.local` ファイルを作成し、Gemini API キーを設定してください。

```bash
GEMINI_API_KEY=your_api_key_here
```

### 3. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開き、すぐに使い始めることができます。
